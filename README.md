# 5-in-1 Voice-Controlled Games

A collection of **five classic arcade-style games** developed in **C++ using SFML**, featuring a **voice-controlled input system** powered by Python.  
The project demonstrates real-time game mechanics, state management, and integration between gameplay systems and external input models.

A gameplay **showreel** is included to showcase voice-based interaction and in-game responsiveness.
